# Slot-Game

Proyek `Slot-Game` adalah simulasi mesin slot klasik sederhana yang diimplementasikan dalam Python. Game ini berjalan melalui antarmuka konsol/terminal, memungkinkan pengguna untuk memasang taruhan, memutar gulungan, dan melihat hasilnya.

* **Simulasi Putaran Acak:** Menggunakan modul `random` Python untuk menghasilkan hasil putaran gulungan yang tidak dapat diprediksi.
* **Sistem Saldo & Taruhan:** Mampu melacak saldo pemain, menerima taruhan per putaran, dan mengurangi/menambahkan saldo sesuai hasil.
* **Perhitungan Kemenangan:** Logika internal untuk mengevaluasi kombinasi simbol pada baris pembayaran dan menentukan jumlah kemenangan.
* **Antarmuka Konsol Sederhana:** Interaksi dengan pengguna dilakukan melalui input/output teks di terminal.
* **Simbol Kustom:** Menggunakan array atau daftar simbol yang berbeda untuk variasi permainan.
